# Code of Conduct

## Our Pledge

We pledge to make participation in this project a welcoming and harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Focus feedback on the code and the project.
- Accept reasonable differences of opinion.
- Help keep discussions useful to contributors and users.

## Unacceptable Behavior

Harassment, personal attacks, discrimination, threats, and publication of private information are not acceptable.

## Enforcement

Report conduct concerns privately to the maintainers through the repository owner's GitHub profile. Maintainers may remove comments, reject contributions, or restrict participation when necessary.
